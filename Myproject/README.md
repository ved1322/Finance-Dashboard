# Finance Dashboard

## Features
- Dashboard with income, expenses, balance
- Transactions with add/edit
- Search and filter
- Role-based UI (Admin / Viewer)
- Insights with analytics
- Local storage persistence
- Dark mode

## Tech Used
- React (Vite)
- CSS

## How to Run
npm install  
npm run dev

## Notes
This project demonstrates frontend UI, state management, and user interaction.
